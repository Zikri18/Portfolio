2D Animation Group Project.

Project Overview

2D Animation Group Project is a project that test me and my group ability to utilise all the knowledge that we have learn. Such as utilising Adobe Photoshop, Adobe Character Animate, Adobe Animate CC, Aobe Premier Pro and Audacity. Upon using this software, I had gain some knowledge on how to improve my skills in editing.

The project were assign to us for 1 month. The project consist of 1 main editor and 2 sub editor helping in creating the scene required for our mini trailer. The project are split into 4 different role for 4 students which are the leader, report writer, voice actor and editor. All 4 of us each done our respective job to create the mini trailer for our group project.

The purpose of this project is to create a mini trailer from any malay folklore that are then needed to be animated with the duration of 3 minute. Our group pick The Legend of Tanggang in which we created the scenes base on the story and make it a bit dramatic by adding some sfx to make the environemnt more believable.
