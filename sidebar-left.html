Claymation Group Project.

Project Overview

Claymation Group Project is a project that test me and my group on how to make a stop motion video with clay. The project were create to test our ability and creativity on capturing motion and implement some of the 12 principle of animation in the creation of the video.

The project were assign to us for 3 month. The project are split into several role for 4 students which are the leader, report writer, voice actor, script writer, video editor and graphic editor. All 4 of us each done our respective job to create a claymation video for our group project.

The purpose of this project is to create a video of claymation using various types of clay and principle of animation to make a smooth motion with various editing software
